# Aron Test Math Quiz Website

A simple responsive math quiz website built with HTML, CSS, and JavaScript. It is designed for Grade 7 and 8 students and includes 50+ multiple-choice questions with instant feedback.

## ⭐ Features

- 50+ predefined Grade 7/8 math questions
- Randomized question selection without repetition
- Instant validation of answers (correct/wrong) with color-coded feedback
- Score tracking (correct / attempted)
- Modern, fun UI with gradient theme and animated buttons
- Mobile-friendly responsive layout

## 🧩 File structure

- `index.html` - main page markup
- `style.css` - styling and visual theme
- `script.js` - quiz logic and interactivity
- `README.md` - project description and usage

## 🚀 Usage

1. Open `index.html` in a web browser.
2. Read the question and select an answer.
3. Result appears instantly in the status box.
4. Click **Next Question** to continue.

## 🛠️ Customize

- Add/remove questions in `script.js` under the `questionPool` array.
- Change visual style in `style.css`.
- Update text in `index.html`.

## 🙌 Notes

- The quiz automatically prevents repeating questions until all have been used.
- After all questions are answered, a completion message appears.
